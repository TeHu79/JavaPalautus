Tervetuloa sivuillemme!

Tee web toteutus, jossa node.js palvelinsovellus lähettää sitä kutsuvalle selaimelle kuvitteellisen yrityksen sivuston html, css ja selaimessa ajettavan javascript koodin

Sivustossa on pääsivu, yritysesittely, yhteystiedot ja henkilökunnan tiedot min 3-5 työntekijää

Pääsivulla on yrityksen yleistä tietoa sekä linkit / napit muihin näkymiin

Html dokumentissa on yksi DIV, johon muiden sivujen sisältö tuodaan GetElementById:n perusteella.

Kun kutakin sivua kutsuvaa painiketta/linkkiä klikataan, tuodaankin esim. divi alue -kohtaan kulloinkin vaadittava tieto JavaScriptin avulla kovakoodatusta string muuttujasta joka sisältää html tägit.

Sivulle tarvittava sisältö rakennetaan DOM-päivityksellä

Todellisia/fyysisiä HTML -sivuja/tiedostoja on VAIN YKSI jolloin sivulta toiselle siirtyminen on todella nopeaa

Tuo henkilökunnan tiedot taulukkoon json tiedostosta siten että kun henkilökunnan tietosivulle mennään selainkoodi samalla kutsuu palvelinta pyytäen json datana henkilökunnan tiedot ja vastauksen saatuaan kirjoittaa sen taulukkoon. Tässä simuloidaan selaimen näkökulmasta sellaista tilannetta että tiedot tuotaisiin back-endin kautta tietokannasta ja lähetettäisiin jsonina, vaikka back-endin osalla oikaistaankin siinä, että luetaan ko. data vain .json päätteisestä tiedostosta eikä todellista tietokantayhteyttä ole, mutta selainkoodi ei sitä tiedä vaan se pyytää dataa ja saa dataa node sovellukselta.
